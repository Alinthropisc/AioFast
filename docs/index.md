# ⚡ AioFast

**AioFast** is an async-first Python framework that combines a Laravel/Django-style
developer experience with the performance of [Litestar](https://litestar.dev) (HTTP/API)
and [Aiogram](https://aiogram.dev) (Telegram bots).

## Highlights

- **IoC container & service providers** — automatic dependency injection.
- **`aiocraft` CLI** — `serve`, `key:generate`, `route:list`, and `make:*` generators.
- **Controllers & routing** — decorator-based controllers compiled to Litestar handlers.
- **Database** — SQLAlchemy models, migrations, seeders, factories.
- **Auth, validation, logging, config** — batteries included, fully typed.
